# Miden Handbook

A community guide to understanding Miden — privacy for real finance.

## Deployment to Render

This is a static HTML website ready to deploy to Render.

### Quick Deploy Steps:

1. **Connect to Render:**
   - Go to [render.com](https://render.com)
   - Click "New +" → "Static Site"
   - Connect your GitHub repository

2. **Configure the deployment:**
   - **Build Command:** Leave empty or use `echo "No build needed"`
   - **Publish Directory:** `.` (current directory)
   - Click "Create Static Site"

3. **Done!** Your site will be live at your Render URL.

### Local Preview

Simply open `index.html` in your browser to preview locally.
